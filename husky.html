哈士奇眼控叫人鈴基本設定

⚙️ 叫人鈴基本設定步驟

  1. 將 HUSKYLENS2 開機，進入系統設置 (System Settings) ➔ Language ➔ 中文 (繁體)。
  2. 於系統設置 (System Settings) ➔ 協議種類 (Protocol Type) ➔ I2C。
  3. 選擇注視方向檢測 (Eye Gaze)，進行眼控檢測。
  4. 第一次使用注視方向檢測 (Eye Gaze) 時，需要先讓設備進行學習。
  5. 按下按鈕 A 進行學習，每個 ID 學習至少 10 張圖。
  6. 學習順序：眼球向左看、眼球向右看、眼球看鏡頭、眼球看左上或左下、眼球看右上或右下。
    (※ 注意：學習與使用時頭與鏡頭、鏡頭距離皆需固定保持不動)。

哈士奇眼控叫人鈴規則

📜 指令兩層式安全防誤觸規則

指令順序分為兩層，用以有效避免日常眼球晃動所產生的誤觸：

第一層：啟用指令 (解鎖)

  • 指令順序：ID 4 (眼球向左下或左上看) ➔ ID 5 (眼球向右下或右上看)。
  • 每一個 ID 的注視時間必須持續 1 秒才算輸入完成。
  • 完成第一層指令後，靜候 1 秒進入第二層。

第二層：控制指令 (叫人鈴)

  • 指令一：ID 1 (眼球向左看) ➔ ID 1 (眼球向左看) ➔ ID 1 (眼球向左看)。
  • 指令二：ID 1 (眼球向左看) ➔ ID 2 (眼球向右看) ➔ ID 1 (眼球向左看)。
  • 每一個 ID 的注視時間持續 0.5 秒即輸入完成。
  • 輸入完成後，靜候 1 秒即會觸發叫人鈴。

⚠️ 注意事項

  • 第一層指令輸入第一個 ID 後，若在 10 秒內未完成第一層指令，將會自動退出。
  • 進入第二層指令後，若在 50 秒內沒有輸入完成，將會自動退出。
  • 裝置上的 MIC/RGB 燈顯示綠燈時，表示鏡頭成功辨識並抓取到學習過的 ID。
